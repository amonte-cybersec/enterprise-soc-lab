\# VMware Workstation Pro Installation



\## Objective



Install and configure VMware Workstation Pro as the virtualization platform for the Enterprise SOC Lab environment.



\---



\## Purpose



VMware Workstation Pro will provide the platform used to deploy and manage the virtual machines required for the SOC simulation environment.



Planned virtual machines include:



\- ELK SIEM

\- Windows Server Domain Controller

\- Windows 11 Employee Endpoint

\- Ubuntu Linux Server

\- Kali Linux Attacker

\- pfSense Firewall
