\# Enterprise SOC Lab - Lab Objectives



\## Objective



The objective of this project is to design, build, secure, monitor, and document a simulated enterprise environment that reflects common Security Operations Center (SOC) workflows.



\---



\## Technical Objectives



Throughout this project I will:



\- Deploy a virtual enterprise infrastructure using VMware Workstation Pro

\- Configure segmented corporate and attacker networks

\- Build an Active Directory domain environment

\- Deploy Windows and Linux systems

\- Centralize logs using the ELK Stack

\- Configure pfSense as the network firewall

\- Implement Suricata for network intrusion detection

\- Perform vulnerability assessments

\- Simulate cyber attacks from Kali Linux

\- Investigate alerts and security events

\- Develop detection techniques and investigative workflows

\- Document each phase of the project with technical evidence



\---



\## Professional Objectives



This project is intended to strengthen practical experience in:



\- Security Operations (SOC)

\- Blue Team Operations

\- Network Security

\- Windows Administration

\- Linux Administration

\- Detection Engineering

\- Threat Hunting

\- Incident Response

\- Technical Documentation



\---



\## Success Criteria



The project will be considered successful when the environment is capable of:



\- Generating security events

\- Collecting and storing logs

\- Detecting malicious activity

\- Investigating security incidents

\- Documenting findings with repeatable procedures
