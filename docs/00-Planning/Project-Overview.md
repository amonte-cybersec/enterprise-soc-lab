\# Enterprise SOC Lab - Project Overview



\## Project Purpose



The Enterprise SOC Lab is a hands-on cybersecurity project designed to simulate a real-world Security Operations Center (SOC). The environment is built to demonstrate practical experience in enterprise infrastructure, security monitoring, threat detection, incident response, and defensive security operations.



Rather than focusing on individual tools, this project emphasizes how security technologies work together to protect an enterprise network from modern cyber threats.



\---



\## Project Goals



The primary goals of this lab are to:



\- Design a realistic enterprise network

\- Deploy Windows and Linux systems

\- Build an Active Directory environment

\- Implement centralized log collection

\- Detect malicious activity using security monitoring tools

\- Investigate security events

\- Document each phase of the deployment and analysis process



\---



\## Enterprise Architecture



The lab consists of two isolated network segments.



\### Corporate Environment



\- ELK SIEM

\- Windows Server Domain Controller

\- Windows 11 Employee Workstation

\- Ubuntu Linux Server

\- pfSense Firewall

\- Suricata IDS



\### Attack Environment



\- Kali Linux Attacker



The two environments allow attacks to be generated safely while security monitoring tools detect and investigate the activity.
