\# Enterprise SOC Lab



\## Overview



This project documents the design, deployment, and operation of a simulated enterprise Security Operations Center (SOC) environment.



The lab focuses on security monitoring, detection engineering, threat hunting, incident response, and phishing analysis.



\## Objectives



\- Build a simulated corporate security environment

\- Deploy centralized security monitoring solutions

\- Collect and analyze endpoint and network telemetry

\- Develop detection and investigation workflows

\- Document incident response procedures



\## Lab Architecture



The environment simulates a corporate network consisting of:



\- Security Information and Event Management (SIEM)

\- Active Directory Domain Controller

\- Windows Employee Endpoint

\- Linux Server

\- Firewall and IDS/IPS monitoring

\- Attacker environment
